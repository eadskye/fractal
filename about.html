About

FractalShop is a fractal generator designed to create a fun environment for the user to explore the visual effects of the Mandelbrot Set. The fractal was generated from a library called Rosetta Code and implements a randomized quote API on the Zoom buttons. The idea for the project came from experimentation with the Mandelbrot Set code written in Javascript, and how to generate the fractal in a browser. Mathematics credit: Dr. Jerome Barre.
